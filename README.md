# Vrinda Store Sales Dashboard

## Overview

An interactive Excel dashboard created to analyze Vrinda Store's sales performance. The dashboard provides insights into sales trends, customer demographics, order status, sales channels, product categories, and top-performing states.

## Features

* Interactive slicers for Month, Channel, and Category
* Sales and order trend analysis
* Gender-wise customer analysis
* Order status tracking
* Top-performing states visualization
* Age and gender distribution analysis
* Sales channel performance comparison

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Formatting
* Excel Functions

## Key Insights

* Women contribute the majority of sales.
* Most orders are successfully delivered.
* Amazon is the leading sales channel.
* Maharashtra generates the highest sales among all states.

## Outcome

This project demonstrates Excel dashboarding, data analysis, visualization, and reporting skills commonly used in Data Analyst roles.
